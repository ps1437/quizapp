Quiz App jar running instruction
      •	Open CMD  ( Win+R  and type cmd)
      •	Write the below line in cmd
        Java  –jar  <jar location> <inputFile location> <output file location>
   
         output file location – optional ( use to see the set of question )
        

  EG : java –jar  C:\QuizApp.jar D:\input.txt E:\output.txt(optional)
      • Hit the enter button

Note : output.txt is optional 
      It is used to print the set of 10 question in a .txt file
